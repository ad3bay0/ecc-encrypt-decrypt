# Elliptic Curve Cryptography - Encrypt & Decrypt (Java)


This demo code includes:
 - ECC private/public key generation to file
 - Encrypt and Decrypt text 

run code: 
``` 
mvn compile exec:java -Dexec.mainClass="com.ad3bay0.crypto.ECC"
```
